# Политика конфиденциальности

Этот файл содержит HTML-страницу с политикой конфиденциальности для приложения "Код Да Винчи: Математик".

## Как использовать

1. **Размещение на GitHub Pages (рекомендуется):**
   - Создайте репозиторий на GitHub
   - Загрузите файл `index.html` в репозиторий
   - Включите GitHub Pages в настройках репозитория
   - Получите URL вида: `https://yourusername.github.io/privacy-policy`

2. **Размещение на другом хостинге:**
   - Загрузите файл `index.html` на любой веб-хостинг
   - Получите прямой URL к файлу

3. **Использование в RuStore:**
   - Укажите полученный URL в разделе "Политика конфиденциальности" при публикации приложения

## Важно

- **Замените** `[Укажите ваш email для связи]` на реальный email адрес в файле `index.html`
- Проверьте, что все ссылки работают корректно
- Периодически обновляйте дату "Последнее обновление" при внесении изменений

## Структура файла

- `index.html` - HTML-страница с политикой конфиденциальности
- `README.md` - инструкция по использованию (этот файл)
